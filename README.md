# DecidIA
Juego interactivo para la toma de decisiones

# DECIDIA - ESPECIFICACIÓN INICIAL DEL PROYECTO

## Descripción General

Desarrollar una plataforma web llamada "DecidIA", diseñada para ayudar a las personas a tomar decisiones mediante dos modos:

1. Modo Gratuito (Decisión por Suerte)
2. Modo Inteligente Premium (Decisión por Análisis e IA)

La experiencia debe ser divertida, moderna, adictiva y altamente visual.

---

# CONCEPTO PRINCIPAL

DecidIA es una plataforma de ayuda para la toma de decisiones.

El usuario realiza una pregunta y puede obtener:

* Una respuesta rápida basada en azar y entretenimiento.
* Una respuesta avanzada basada en lógica, perfil personal e inteligencia artificial.

La experiencia debe sentirse como una mezcla entre:

* Magic 8 Ball
* Máquina tragamonedas
* Gamificación
* Perfil psicológico
* Asistente inteligente

---

# EXPERIENCIA GRATUITA

## Pantalla Principal

Mostrar:

Logo DecidIA

Frase principal:

"¿No sabes qué decidir? Pregúntale a DecidIA"

Campo de texto:

"Escribe tu pregunta"

Ejemplos:

* ¿Debo aceptar este trabajo?
* ¿Le escribo hoy?
* ¿Compro ese producto?
* ¿Hago ese viaje?

Botón principal:

"Consultar"

---

# MECÁNICA TIPO TRAGAMONEDAS

La respuesta gratuita NO debe mostrarse instantáneamente.

Debe utilizar una animación inspirada en las máquinas tragamonedas.

Sin embargo:

NO utilizar 3 columnas.

Utilizar UNA SOLA COLUMNA CENTRAL.

Funcionamiento:

1. Usuario escribe pregunta.
2. Presiona Consultar.
3. La columna comienza a girar verticalmente.
4. Se muestran mensajes rápidos pasando.
5. La velocidad disminuye gradualmente.
6. Finalmente se detiene.
7. Se muestra la respuesta final.

Duración:

Entre 3 y 5 segundos.

Debe generar expectativa.
